This is a simple translator app using Google Translate API and MeCab for Japanese readings (kana/romaji).
To run:
1. Replace the placeholder API key in the code with your actual Google API key.
2. Install MeCab on your Mac:
   brew install mecab mecab-ipadic
3. Build and run in Xcode.

Это простое приложение-переводчик, использующее Google Translate API и MeCab для отображения японского текста (кана/ромадзи).
Как запустить:
1. Замените фейковый API-ключ на свой Google API ключ.
2. Установите MeCab на Mac:
   brew install mecab mecab-ipadic
3. Соберите и запустите в Xcode.

使い方：
